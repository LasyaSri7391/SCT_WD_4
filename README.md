# To-Do Web Application

A clean and responsive To-Do Web Application built using **HTML**, **CSS**, and **JavaScript** to help users organize and manage daily tasks.

## Features

* Add new tasks
* Edit existing tasks
* Mark tasks as completed
* Delete tasks
* Responsive design
* Clean and intuitive interface

## Technologies Used

* HTML5
* CSS3
* JavaScript
